2. Where and how does JavaScript typically run?

JavaScript is running in the browser on the client side, it runs after the server has given the website, and as soon as the website is loaded and it will continue to run if the user keeps on using events like moving mouse, clicking or typing, as JavaScript is listening for these events.

The way the JavaScript runs on a website is by built in JavaScript engines in the browser. The built in engines are also known as the built-in interpreter. Their job is to find the code, read it and execute it. Different browsers have different engines that read JavaScript, for example:

  • Google Chrome uses an engine called V8
  • Mozilla Firefox uses 3 engines these are Gecko layout engine, SpiderMonkey, and Rhino
  • Safari uses an engine called Nitro
